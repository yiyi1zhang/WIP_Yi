Edit result

The flipping images and fitting size functions are not suitable for result images.
  1. Set colors
    The colors of mask(s) are initially set be default colors or the preferred colors saved in result file. If you want to edit the mask colors, you can trigger a color window by clicking the item in the menu Settings. There are pre-defined 2, 8, and 11 classes supported, or the corresponding number of classes showing in the current result file. You can choose DIY to set colors, hatches and transparency in the window and confirm it. In default mode, the modification will not be saved. When you set the color for the last item in DIY mode, you should at first check on the radio button in front of each color button, only then you can select the desired color for the corresponding class.
  2. Show class name
    If there are more than one class in the result image, you can choose to show the name in the drop down box show class name. Then the class name will be around cursor when you click left mouse button in the region. Additionally, there will be also a legend showing the names and colors of different classes at the left top corner. If the legend covers the image, you can click legend on/off to show the legend on/off.
  3. Set transparency You can either set transparency in the color window or on the slider bar Set transparency. The contrast between images and masks will be changed.
