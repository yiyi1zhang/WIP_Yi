Load data

  1. Open images
    At the top of the tool bar, you can select to load images from a path (a directory or a single file). The GUI supports a wide range of image formats to view, including the normal image formats .CUR, .ICNS, .SVG, .TGA, .BMP, .WEBP, .GIF,.JPG, .JPEG, .PNG, .PBM, .PGM, .PPM,.TIFF,.XBM; a single DICOM image, including .IMA, .DCM; a folder with all DICOM images; a single file in .NII; In addition, a single file in.MAT, .NPY where stores image array and image arrays with multi-dimensions.

    In another way, you can also load images from workspace if there are datasets selected in the second tab Training Network. Further, if you select a single item in the datasets window, only that dataset will be shown.

  2. Load result
    You can also show a result file in the format .MAT, .NPY, or .NPZ in the canvas. At first, you should choose to load result from a path or from workspace. If there are images in the data, they will be automatically shown; otherwise, you will encounter a pop up window just like the drop down box Open image, and you can select the matching images with the result file. Then, you need to click the button switch to change the canvas to show results. If you want to switch to another result file existing in the drop down box, please at first go back to the image showing pattern by clicking the switch button, in order to make sure, that the images and results coordinate with each other.
    Figure 1: Show Result
  3. Load previous
    You can also load the previous images you have shown if you save it by triggering the corresponding item in menu File.
