Assistance

There are also some assistant methods to have a deeper insight of image or to label on the image more conveniently.
  1. Show information
    If you want to view all local, global and built-in functions used in this GUI, you can click the button Show Information. Then a window with a drop down box will pop up. And the selected item will be shown in a list in the window.
  2. Select region of interest(ROI)
    Click the button ROI, a independent window will pop up, showing the current image on the screen, you can select the ROI under the image. And you can drag and change the size of a rectangular box to select ROI, and change the contrast and viewing the pixels of ROI.
    Figure 3: Select ROI
  3. Cursor inspector
    If you check on the cursor inspector, the cursor will be surrounded with two cross lines wherever you click left mouse button on the image. If the image is in three dimensional mode, the slices will be changed synchronously to the clicked cursor position.
    Figure 4: Cursor Inspector
